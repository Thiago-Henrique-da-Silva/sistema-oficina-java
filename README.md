# Sistema de Oficina Mecânica

Projeto desenvolvido em Java utilizando:
- Programação Orientada a Objetos (POO)
- Collections (`List` e `Map`)
- Exceptions personalizadas
- Enum
- Estrutura em camadas

## 📌 Estrutura do Projeto
- `dominio` — entidades do sistema (Cliente, Carro, OrdemDeServico)
- `service` — regras de negócio
- `exception` — exceções personalizadas

## 📌 Funcionalidades
- Cadastro de clientes
- Cadastro de carros
- Criação de ordens de serviço
- Atualização de status da ordem
- Listagem de ordens de serviço
- Busca de cliente por CPF
- Busca de carro por placa

## 📌 Status da Ordem
- `INICIADO`
- `CANCELADO`
- `FINALIZADO`

## 📌 Como rodar
- Java 17 ou superior
- Clone o repositório e execute a classe `Main`
